About This Project

This application allows users to use style transfer on their own images. Try it Out

What is Style Transfer?

It's used to transfer the style of one image onto another. In this application, the style of artwork is transfered on your self or other provided image.

How does it work?

A convolutional neural network is trained to produce images right in between the Content Image and the Style Image.

This application uses an existing implementation which can be found here.
