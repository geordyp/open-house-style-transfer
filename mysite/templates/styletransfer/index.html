About the Project

We've created an application which allows users to use style transfer on their own images.

What is Style Transfer?

It's used to transfer the style of one image onto another. In our application, we transfer the style of artwork onto selfies.

How does it work?

A convolutional neural network is trained to produce images somewhere between the Content Image and the Style Image.

This application uses an existing implementation which can be found here.
